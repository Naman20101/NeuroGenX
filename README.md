# NeuroGenX — AI that evolves itself

> **Tagline:** "AI that builds smarter AIs."

NeuroGenX is an experimental, next-generation AI framework designed to **automatically generate, test, and improve its own AI agents** — enabling rapid evolution of capabilities without manual coding for each update.

---

## 🚀 Vision
To create an autonomous AI ecosystem where intelligent agents **continuously learn, adapt, and create new AI models** — unlocking innovation at a scale beyond human speed.

---

## 🧠 Core Features (Planned)
- **Self-Improving AI Agents** — Agents that design, train, and test new AI architectures.
- **Multi-Agent Collaboration** — Different AIs specializing in vision, language, reasoning, etc., working together.
- **Dynamic Knowledge Graph** — Continually updated web of concepts and facts across domains.
- **Web + API Interface** — Control NeuroGenX from a clean dashboard or API calls.
